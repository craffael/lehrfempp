Detailed Description

Represents a physical entity as defined in gmsh. In GMSH a Physical entity is created through one of the commands Physical Point, Physical Line, Physical Surface or Physical Volume and represents a collection of points, lines, surfaces or volumes.

Every Physical Entity has:

  • A dimension (of the entities that it contains)
  • A Number which must be unique per dimension, i.e. a point and a line can have the same number
  • A name (string)
