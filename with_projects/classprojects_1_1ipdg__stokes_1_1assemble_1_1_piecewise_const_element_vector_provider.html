Detailed Description

Element vector provider for the stokes system.

Note
Currently, only triangular meshes are supported

The linear form is given by

\[ v \mapsto \int_K\! f \cdot v \,\mathrm{d}x + \sum_{e\in\mathcal{E}^\partial(\mathcal{M})} \int_e\! \frac{\sigma}{|e|} \left(g_D\cdot\tau_e\right) \left[\!\left[v\cdot\tau_e\right]\!\right] \,\mathrm{d}x \]
