Detailed Description

Local assembler for a matrix mapping boundary basis function coefficients to jumps.

This assembler builds a matrix mapping the basis function coefficients f nodes on the mesh boundary to the normal component of the jump over the boundary edges. This is used for assembling the offset function, as the dirichlet data is given in terms of boundary jumps, but the actual solution uses the nodal values as basis function coefficients.
