Detailed Description

An elment matrix provider computed with the whitney one form, a vector valued basis function.

The element matrix provider works in a 3 dimensional world with 2 dimensional triangular cells.

\[ (\mathbf{u}, \mathbf{v}) \mapto \int\limits_K \text{curl}_{\Gamma}(\mathbf{u}) \ \text{curl}_{\Gamma}(\mathbf{v}) dx \]

The whitney 1-forms, surface edge elements are associated with edges and defined as

\[ b_i = s_i (\lambda_i \mathbf{grad}_{\Gamma}(\lambda_{i+1}) - \lambda_{i+1} \mathbf{grad}_{\Gamma}(\lambda_{i})) \]

Note
This class complies with the type requirements for the template argument ENTITY_MATRIX_PROVIDER of the function lf::assemble::AssembleMatrixLocally().
Only triangular meshes are supported
