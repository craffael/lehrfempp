Detailed Description

Element matrix provider for rotated whitney one form.

\[ (\mathbf{v}, u) \mapsto \int\limits_K \text{div}_{\Gamma}(\mathbf{v})\, u \ dx \]

The element matrix provider works in a 3 dimensional world with 2 dimensional triangular cells.

As basis functions, the rotated Whitney 1-forms, surface edge elements are used for \(\mathbf{v}\) and the cellwise constant basis functions are used for u

The whitney 1-forms, surface edge elements are associated with edges and defined as

\[ \mathbf{b}_i = s_i (\lambda_i \mathbf{grad}_{\Gamma}(\lambda_{i+1}) - \lambda_{i+1} \mathbf{grad}_{\Gamma}(\lambda_{i})) \]

with \( \lambda_i \) barycentric basis functions and \( s_i \) is a sign of the function based on the relative orientation of the edge in the mesh.

The element matrices are equivalent to the element Matrices for

\[ (\mathbf{v}', u) \mapsto \int\limits_K \text{curl}_{\Gamma}(\mathbf{v}')\, u \ dx \]

with the normal whintey one basis functions for \(\mathbf{v}'\)

Note
This class complies with the type requirements for the template argument ENTITY_MATRIX_PROVIDER of the function lf::assemble::AssembleMatrixLocally().
Only triangular meshes are supported
